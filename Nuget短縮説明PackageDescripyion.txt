CommonBatchFramework is a minimal framework for C# batch processing applications.

It standardizes common patterns such as execution flow, directory structure, exception handling, and logging, without introducing unnecessary features or dependencies.

Key components:

* AppRunner: unified entry point with error handling and lifecycle control
* GlobalPaths: centralized path management based on AppContext.BaseDirectory
* Log: consistent logging to console and output file

The framework focuses on defining rules rather than providing features, enabling developers to build consistent and maintainable batch applications across projects.

Suitable for small to medium-scale batch systems and single-executable deployments.
